# 소나기 — 황순원
학습용 웹페이지 · เว็บเพจประกอบการเรียน

## Pages
- `index.html` — เนื้อเรื่อง 소나기 พร้อมคำศัพท์และคำถาม
- `dokhugan.html` — ใบงานเขียน 독후감

## Deploy

### GitHub Pages
1. Push โฟลเดอร์นี้ขึ้น GitHub repository
2. ไปที่ Settings → Pages
3. เลือก Source: `main` branch, folder: `/ (root)`
4. กด Save → รอแป๊บเดียวก็ใช้งานได้

### Vercel
1. Import repository จาก GitHub เข้า Vercel
2. Framework Preset: **Other**
3. กด Deploy ได้เลย ไม่ต้องตั้งค่าอะไรเพิ่ม

## URLs หลัง deploy
- `/` → หน้าเนื้อเรื่อง
- `/dokhugan` → ใบงานเขียน독후감
